Hay cuatro tipos-> siempre tienen dos partes

Zero -> si sale el sol, el hielo se derrite -> if the sun rises, the ice melts -> es lo que pasa si o si, ley indiscutible -> las dos partes en presente

First -> if siempre con el simple present, la otra con el futuro con will -> si ellos me invitan, voy -> if they invite me to the party, I will go -> es una posibilidad real, algo que puede pasar 

Second -> if con simple past, la segunda con would y el verbo en infinitivo -> lo climático se usa con este -> if it rained, I would stay at home -> aca se permite usar I were -> es una posibilidad irreal -> If I were you, I would -> Tiene que ver con el futuro

Third -> if con past perfect, la segunda con would y presente perfecto -> siempre se usa have, incluso si hablamos de she
